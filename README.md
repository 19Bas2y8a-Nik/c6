# Калькулятор для Vercel

## Описание

Это одностраничный браузерный калькулятор, готовый к деплою на Vercel.

## Файлы
- `calculator.html` — основная страница калькулятора.
- `vercel.json` — конфиг для корректного роутинга на Vercel (чтобы по адресу `/` открывался калькулятор).

## Деплой на Vercel

### 1. Убедитесь, что у вас есть аккаунт на Vercel (https://vercel.com)

### 2. Загрузите (или синхронизируйте) этот проект в ваш репозиторий GitHub/GitLab/Bitbucket.

### 3. Перейдите на https://vercel.com/import и выберите ваш репозиторий (или выберите "Import Project" в интерфейсе Vercel).

### 4. Когда Vercel спросит настройки:
- **Framework Preset**: `Other`
- **Root Directory**: `/<ваша_папка_проекта>` (или оставьте `/`, если проект лежит в корне)

### 5. Жмите "Deploy"

### 6. После публикации — калькулятор будет доступен по корневому адресу вашего Vercel-проекта.

### 7. Проверяйте ссылку:
```
https://<your-vercel-project>.vercel.app/
```

---

#### Пример быстрой локальной проверки (откроет калькулятор в браузере):
```powershell
start calculator.html
```
